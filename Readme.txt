###  RFM Segmentation in Online Retail Dataset
This project focuses on RFM (Recency, Frequency, Monetary) segmentation analysis in an online retail dataset. RFM segmentation is utilized to analyze customer groups based on their loyalty levels. The code provided in this repository includes data preprocessing steps, handling missing values, and performing RFM segmentation analysis.


###   Results
The results of the RFM segmentation analysis will be presented in the results directory. This includes visualizations such as RFM scatter plots and segmented customer groups based on loyalty levels.
